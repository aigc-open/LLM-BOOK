欢迎来到 NVIDIA cuTile Python 执行模型的深度解析。

cuTile Python 是 NVIDIA 推出的新一代 GPU 编程框架，它为 GPU 编程带来了全新的抽象层次。

在本系列视频中，我们将深入讲解 cuTile 的执行模型，帮助你理解代码是如何在 GPU 上运行的。

cuTile 的三大核心特性包括：

第一，Block 级并行。你可以显式指定网格和线程块的组织方式，获得对并行执行的精确控制。

第二，Tile 抽象。以数据块为核心的高层抽象，让你无需关心底层线程管理的复杂细节。

第三，自动同步。系统会自动处理线程同步问题，你不再需要手动调用同步操作。

理解执行模型是编写高效 GPU 程序的关键，让我们开始这段学习之旅吧！
